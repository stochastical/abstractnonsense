This site is built with the [Hugo](https://github.com/gohugoio/hugo) static site generator and served by [Cloudflare Pages](https://pages.cloudflare.com).

Math formula typesetting is rendered by [MathJax](https://github.com/mathjax/MathJax-src). I prefer MathJax to [KaTeX](https://github.com/KaTeX/KaTeX) as it is both accessible and produces results closer to LaTeX. The downside is that it loads a large JavaScript blob from a CDN and incurs page reflows as it injects rendered SVGs in place of the inline LaTeX formulas.

For analytics, I use [Umami](https://github.com/umami-software/umami), which is anonymised, cookie-less and privacy-preserving.

The source code is available on [GitHub](https://github.com/stochastical/abstractnonsense). When I write about more substantive changes pertaining to this site, I'll tag the post with [`abstractnonsense.xyz`](/tags/abstractnonsense.xyz/).
